# Calculator-Using-JAVA
A basic Calulator Using JAVA AWT,Swing and Event Handling

$Mini Java Calculator$

This is a simple calculator application made using Java Swing. It performs basic arithmetic operations like addition, subtraction, multiplication and division.

## About the project

This project is created to understand how GUI works in Java using Swing and AWT. It uses GridBagLayout to arrange buttons and handles user actions using ActionListener.

## Features

* Basic operations: addition, subtraction, multiplication and division
* Clear button to reset input
* Simple user interface
* Button click based input

## Technologies used

* Java
* Swing
* AWT

## How to run

1. Download or clone the project
2. Open terminal in the project folder
3. Compile the program
   javac Calc.java
4. Run the program
   java Calc

## How it works

* Enter a number using the buttons
* Click an operator
* Enter the second number
* Press = to see the result
* Press C to clear everything

## Limitations

* Does not handle division by zero
* No advanced operations
* Limited input validation
